# Balanced Pitch
Modern one-page website for small and medium-sized businesses.
The project is implemented with an emphasis on simplicity, functionality and adaptability in order to
the site looked great and worked on all devices.

Balanced Pitch — современный одностраничный сайт для малого и среднего бизнеса.
Проект реализован с акцентом на простоту, функциональность и адаптивность, чтобы
сайт отлично выглядел и работал на всех устройствах.

**Используемые технологии:**

- React
- Vite
- React Router DOM
- GSAP
- Framer Motion
- Lenis и @studio-freight/react-lenis
- ESLint

The site is easy to scale and supports smooth animations to enhance
the user experience.

Сайт легко масштабируется и поддерживает плавные анимации для улучшения
пользовательского опыта.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md)
  uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)
  uses [SWC](https://swc.rs/) for Fast Refresh
