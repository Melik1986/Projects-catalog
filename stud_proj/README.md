# Каталог товаров Briaton

Веб-приложение-каталог для демонстрации и фильтрации товаров с возможностью добавления в корзину.

## Назначение
Сайт предназначен для малых и средних предприятий, которым нужен современный каталог товаров с удобной навигацией и возможностью оформления заказа.

## Особенности
- Каталог товаров с фильтрами и поиском
- Корзина для оформления заказа
- Современный дизайн и адаптивная верстка
- Использование SVG-иконок и оптимизированных изображений
- Валидация форм и интерактивные элементы
- Быстрая загрузка страниц

## Используемые технологии
- HTML5, CSS3 (SCSS)
- JavaScript (модульная структура)
- Сторонние библиотеки для валидации и слайдеров

## Для кого
Для компаний, которым нужен простой и функциональный онлайн-каталог товаров без сложных CMS и фреймворков.
