## Introduction

This is a static product catalog designed for small and mid‑size businesses. It offers filtering, search, and a simple cart to collect items before checkout.

## Features

- Product catalog with filters and search
- Cart for order preparation
- Modern, responsive layout
- SVG icons and optimized images
- Form validation and interactive UI elements
- Fast loading static pages

## Tech Stack

- HTML5, CSS/SCSS
- Vanilla JavaScript (modular)
- Lightweight helper libraries (validation/slider)
