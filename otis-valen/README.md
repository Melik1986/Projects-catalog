# Otis Valen Portfolio

Веб-сайт-портфолио для творческого специалиста (дизайнера, фотографа или художника), демонстрирующий работы и предоставляющий информацию о себе и услугах.

## Назначение
Сайт служит персональной витриной работ, позволяет посетителям ознакомиться с проектами, узнать контактную информацию и получить представление о стиле автора.

## Особенности
- Несколько тематических страниц: главная, проекты, о себе, контакты
- Современный минималистичный дизайн
- Адаптивная верстка для мобильных и десктопных устройств
- Быстрая загрузка страниц
- Простая навигация между разделами
- SEO-оптимизация для поисковых систем

## Используемые технологии
- HTML5, CSS3 (включая современные подходы к адаптивности)
- JavaScript (структурирован в отдельной папке js)
- Vite для сборки и оптимизации
- Статическая генерация страниц

## Для кого
Для творческих специалистов, желающих быстро запустить современный сайт-портфолио без лишних зависимостей и сложных фреймворков.
