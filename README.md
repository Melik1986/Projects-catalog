## üöÄ **Getting Started**

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Melik1986/Projects-catalog.git
   cd Projects-catalog
   ```

2. **Choose a project** and navigate to its directory:
   ```bash
   cd [project-name]
   ```

3. **Install dependencies**:
   ```bash
   npm install
   # or
   yarn install
   ```

4. **Start development server**:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Read the documentation** in each project's `README.md` for detailed setup instructions.
