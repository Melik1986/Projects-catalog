# Snapgram — социальная сеть для портфолио

Snapgram — современное одностраничное приложение (SPA) для обмена постами, реализованное на React, TypeScript и TailwindCSS.

## Основные возможности
- Регистрация и аутентификация пользователей (Appwrite)
- Публикация, редактирование и удаление постов
- Просмотр ленты, поиск и фильтрация пользователей
- Сохранение избранных публикаций
- Приватные и публичные маршруты
- Быстрый и адаптивный интерфейс

## Технологии
- React 18, TypeScript
- TailwindCSS
- Appwrite (бэкенд, хранение данных и файлов)
- React Query (управление состоянием и запросами)
- React Router DOM
- Zod (валидация форм)
- Radix UI, Lucide Icons

## Преимущества
- Чистая архитектура и модульная структура
- Использование современных библиотек и хуков
- Адаптивная верстка для всех устройств
- Линтеры и форматирование кода (ESLint, Prettier)

## Структура проекта
- `/src` — компоненты, страницы, хуки, контекст, утилиты
- `/public` — ассеты и иконки
- `package.json` — зависимости и скрипты

Проект отлично подходит для демонстрации навыков фронтенд-разработки, работы с современным стеком и построения функциональных SPA.
