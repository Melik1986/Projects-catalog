Snapgram — современное социальное приложение, реализованное на React, TypeScript и TailwindCSS. В проекте используются Appwrite для бэкенда, React Query для управления данными, реализована аутентификация, публикация и редактирование постов, поиск пользователей, сохранение избранного, адаптивная верстка и быстрый UI. Проект демонстрирует навыки построения SPA с приватными и публичными маршрутами, работу с формами, валидацией (Zod), кастомными хуками и современными библиотеками для фронтенда.
