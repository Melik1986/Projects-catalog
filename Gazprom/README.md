# Gazprombank.tech — лендинг раздела вакансий

Лендинг для раздела вакансий Газпромбанк.тех. Проект выполнен с акцентом на минимализм, адаптивность и современный UI. Использованы SCSS, SVG-спрайты, кастомные шрифты, реализовано мобильное меню и оптимизированные иконки.

## Основные секции
- Шапка
- Главный экран
- Мотивация
- Подвал

## Особенности
- Чистый и быстрый код
- Современный дизайн
- Адаптивная верстка
- Оптимизация для мобильных устройств

## Технологии
- HTML5
- SCSS
- SVG-спрайты
- JavaScript (минимально)

Проект демонстрирует умение создавать современные, быстрые и адаптивные сайты для бизнеса.
