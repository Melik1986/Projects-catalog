## Introduction

This project is a careers section landing page for Gazprombank.tech. The design focuses on clarity, speed, and mobile‑first usability. The mobile menu is built with CSS animations and minimal JavaScript, while icons are optimized into a single SVG sprite.

## Features

- Responsive layout with mobile‑first approach
- Animated burger and full‑screen mobile menu
- SVG sprite for all icons (one request, easy theming)
- Custom webfont with `font-display: swap`
- Clean BEM‑style SCSS structure

## Tech Stack

- HTML5
- SCSS
- SVG Sprites
- Vanilla JavaScript (only for menu toggle)
